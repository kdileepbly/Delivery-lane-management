## One-line pitch

A professional Django + React starter kit for building project management dashboards, internal tools, and client delivery platforms.

## Short description

TaskForge gives you a polished full-stack foundation with JWT auth, workspace dashboards, task boards, analytics, Docker setup, and demo data so you can launch faster or showcase production-style engineering on GitHub.

## What is included

- Django REST backend
- React TypeScript frontend
- JWT authentication
- Dashboard analytics
- Task management board
- Demo seed data
- Docker configuration
- Documentation pack
